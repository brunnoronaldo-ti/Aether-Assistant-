# Aether-Assistant

Este repositório é o núcleo de desenvolvimento de um **Assistente de IA Modular** focado em automação de tarefas e análise de dados com Python.

O objetivo principal é unir o poder de processamento do **Pandas** com a inteligência de LLMs (Large Language Models) para criar um agente capaz de automatizar fluxos de trabalho manuais.

## 🚀 Objetivos do Projeto

- [ ] Criar uma interface básica de chat (Streamlit).
- [ ] Integrar automação de arquivos e planilhas (Pandas/OS).
- [ ] Implementar um "Cérebro" via API (OpenAI/Ollama).
- [ ] Desenvolver módulos de Web Scraping (Selenium/Playwright).
- [ ] Tornar a lógica reutilizável para integração em sistemas maiores.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python 3.10+
- **Manipulação de Dados:** Pandas, NumPy
- **IA/Agentes:** LangChain / OpenAI API
- **Automação:** Selenium, Requests
- **Interface:** Streamlit (provisório)

## 📁 Estrutura do Repositório

- `/modules`: Funções isoladas de automação (scraping, filtros, etc).
- `/data`: Arquivos de exemplo (CSV, Excel).
- `/core`: Lógica principal do agente de IA.
- `.env.example`: Modelo de variáveis de ambiente (chaves de API).
